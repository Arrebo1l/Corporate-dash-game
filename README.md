# Corporate Dash: Office Unleashed

Corporate Dash is a Unity-based parkour mini-game designed for working professionals. The game focuses on short, fast-paced gameplay sessions that help players release work-related stress in a fun and interactive way.

## Project Overview
This project explores how casual mobile games can serve as stress-relief tools for people working in high-pressure environments. By combining parkour mechanics, time-limited challenges, and simple combat elements, the game allows players to engage in quick gameplay sessions anytime and anywhere.

## Game Concept
Players control a character navigating office-themed environments while:
- Dodging obstacles
- Collecting coins
- Purchasing props
- Defeating a final boss within a time limit

The game emphasizes accessibility, fast feedback, and short play sessions tailored to busy working professionals.

## Design and Development Process
The project includes:
- Research on workplace stress and player behavior
- Analysis of existing parkour games
- Game concept development and user scenarios
- Scene design and game flow planning
- Unity-based implementation (player controller, timer, props system)
- User testing and iterative design improvements

## Technical Highlights
- Built with Unity
- Player movement and collision detection
- Timer and countdown system
- Coin collection and props purchasing system
- Basic combat mechanics
